# NeroviaFlow – Water Intelligence for Hotels

> Non‑invasive leak detection for hotels. Zero false alarms. From Waste to Wealth – One Drop at a Time.

---

## 📌 Overview

NeroviaFlow is an IoT‑based leak detection system targeting 50‑150 room hotels in water‑stressed regions. Our non‑invasive solution attaches to existing water pipes (no cutting, no plumber) and uses dual‑sensor confirmation (flow + pressure) to eliminate false alarms – a first in the hotel segment.

**Target market:** Casablanca (150+ eligible hotels) → Morocco → Southern Europe & MENA
